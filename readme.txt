== Description ==

=A PLUGIN TO HIDE WORDPRESS NOTIFICATIONS FROM UNAUTHORIZED USERS.=

Notifications Hider is a WordPress plugin designed to improve the security and management of notifications on your site. It allows you to hide administrative notifications from unauthorized users, ensuring that only the relevant people receive critical information.

With Notifications Hider, you can :

- Control who can see WordPress notifications.
- Ensure a cleaner, less cluttered administration experience.
- Improve security by limiting access to sensitive information.

== Languages / Localization ==

Notifications Hider is translated into English and French.

If you'd like to translate the plugin into your language, [please click here](https://translate.wordpress.org/projects/wp-plugins/notifications-hider/)..

== Frequently Asked Questions ==

= Who can use Notifications Hider? =
Notifications Hider is designed for everyone, with no coding skills required.

= What happens if no admin has access to the plugin? =
If no specific admin is selected, all site administrators will have access to the plugin configuration.
